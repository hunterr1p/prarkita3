# prarkita3
Данный проект представляет собой пример рекомендательной системы, реализованной с использованием нейронной сети.  
Система предназначена для рекомендации товаров и услуг на основе пользовательских оценок.  
В рамках учебной практики в качестве примера используется датасет MovieLens, содержащий оценки фильмов, однако применяемый подход может быть использован для рекомендации любых товаров или услуг.

Основная идея проекта заключается в использовании нейронной сети с эмбеддинговыми слоями для пользователей и товаров, что позволяет учитывать скрытые зависимости в данных и формировать персонализированные рекомендации.

Структура проекта
- `rating.csv` — файл с пользовательскими оценками  
- `movie.csv` — файл с информацией о товарах (фильмах)  
- `cod.py` — основной файл с реализацией рекомендательной системы
  
Подготовка к запуску
Убедитесь, что у Вас установлены все необходимые библиотеки, а также файлы movie.csv и rating.csv находятся в той же папке, что и сам код, ибо они нужны для его работы.
